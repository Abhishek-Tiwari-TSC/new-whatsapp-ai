TSC Template Studio

Version 1.0 · Built by Abhishek Tiwari, D2C Engineering · The Sleep Company

What Is This?

TSC Template Studio is an AI-powered internal tool that generates Meta-compliant WhatsApp Business API Utility templates for The Sleep Company. Every outbound message produced conforms to Meta's approval standards — no promotional language, correct placeholder numbering, and proper 4-part structure.

How The System Works

1

Input

Describe the customer touchpoint in plain English.

2

Classification

Engine detects intent and whether input is Utility, Marketing, or Mixed.

3

Sanitisation

Promotional content is stripped; a clean utility brief is extracted.

4

RAG Retrieval

6 semantically similar approved templates retrieved from the vector store.

5

Generation

AI model generates 5 structured variations with auto-fix on failures.

6

Compliance

Banned-word sanitiser and placeholder renumberer runs before output.

Template Structure (Meta Standard)

  1. Context Statement — 1–2 sentences stating what happened. Factual and direct.
  2. Status / Explanation — Timeline, reason, or current status (optional).
  3. Key Details Block — Label: {{1}} format, each field on its own line.
  4. Action / Closing — One clear next step or instruction.

Meta Hard Rules (Auto-enforced)

  • No promotional words: discount, offer, sale, % off, coupon, festive, loyalty, reward, hurry, exclusive, etc.
  • No decorative emojis
  • No exclamation marks
  • No bullet point lists inside templates
  • Placeholders always numbered sequentially: {{1}}, {{2}}, {{3}}
  • One unique number per unique dynamic value — never reuse for a different field

Variation Types

  • Minimal — Fewest fields, shortest. Good for simple confirmations.
  • Specific — More detail: adds date, store, reference number.
  • Action-Oriented — Emphasis on what the customer must do next.
  • Confirmatory — Asks customer to confirm one specific thing.
  • Informational — Most complete; explains status and next steps fully.

Technology Stack

Python 3.11 Flask AI Language Model ChromaDB 0.4 ONNXMiniLM-L6-V2 RAG Pipeline Session Auth

Usage Tips

  • Be specific about the customer action (e.g. "order placed", "return initiated") rather than the message goal.
  • Promotional input is automatically converted — a warning appears when this happens.
  • Click Generate More Variations to get additional alternatives beyond the first 3.
  • Use Generation History to compare and revisit past outputs within your session.
  • Copy any template and paste directly into Meta's WhatsApp Business Manager for submission.
